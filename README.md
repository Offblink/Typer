# Typer - 自动文本输入工具

一款简单易用的自动文本输入工具，可以帮你自动输入预设的文本内容。

## 🚀 快速开始

### 运行程序
直接双击 `Typer.exe` 即可启动程序。

### 基本使用步骤
1. 打开程序后，在"输入文本"区域输入你想要自动输入的内容
2. 根据需要调整设置（默认设置通常已经足够）
3. 点击"开始自动输入"按钮
4. 在倒计时结束前切换到你想要输入文本的目标窗口（如Word、浏览器、记事本等）
5. 程序会自动帮你输入文本！

## ⚙️ 设置说明

| 设置项 | 说明 |
|--------|------|
| 开始前延迟 | 点击开始后等待的秒数，给你时间切换到目标窗口（默认3秒） |
| 字符间延迟 | 每个字符输入之间的间隔时间（默认0秒，即最快速度） |
| 输入方法 | 模拟按键（更快）或逐个字符（更兼容） |
| 重复次数 | 文本重复输入的次数 |

## 📝 输入方法对比

### 模拟按键（推荐）
- **优点**：输入速度快，适合大多数场景
- **适用**：Word、记事本、浏览器表单等普通应用

### 逐个字符
- **优点**：兼容性强，能正确处理回车、Tab等特殊按键
- **适用**：终端、游戏、虚拟机等特殊环境

> 💡 如果遇到回车、Tab等按键无效的情况，建议切换为"逐个字符"模式

## 🆘 帮助文档

程序内置了详细的帮助文档，点击"帮助"选项卡即可查看：
- 应用基本用法
- 输入方法详细说明
- 常见问题解答

## 🖼️ 自定义图标（可选）

如果你想自定义窗口图标，只需将 `icon.ico` 文件放在程序同一目录下即可。如果没有找到图标文件，程序会使用内置的默认图标。

## ⚠️ 注意事项

1. **切换窗口时机**：点击"开始"后，请在倒计时结束前切换到目标窗口
2. **输入法状态**：输入中文时，请确保系统输入法处于中文状态
3. **输入失败处理**：如果输入失败，尝试增加"字符间延迟"（建议0.1-0.5秒）
4. **权限问题**：某些应用可能需要管理员权限才能正常输入
